A privacy-safe, viral share surface (UI-06). On the Result screen — where the deck, drawn cards, and
summary are already loaded — render an offscreen `<canvas>` into an image (deck background from the
per-deck CSS accent/gradient, the 3–4 card faces already fetched for the reveal, the spread name, a
short summary line) and let the user share it via the Web Share API with a download fallback. The
personal question is EXCLUDED by construction — it is simply never drawn. Client-only: no backend

share are noted as future upgrades, not built here.
